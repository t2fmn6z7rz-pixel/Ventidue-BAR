## 📝 Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/t2fmn6z7rz-pixel/VENTIDUE-BAR.git
   cd VENTIDUE-BAR
   ```

2. **Add your logo**
   - Place your frog PNG in `assets/logo.png`

3. **Open locally**
   - Simply open `index.html` in your browser

4. **Deploy**
   - Push to GitHub and enable GitHub Pages
   - Or deploy to Netlify/Vercel by connecting the repository
